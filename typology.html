Загрузка данных WALS...

Примечание к WALS. Таблица загружается из локального файла data/languages.json, построенного по WALS CLDF.

Сноска. Значения WALS показаны в русском переводе, сырой вид сохранён в data/languages.json. Названия семей даны по Glottolog; для крупнейших семей приведены устоявшиеся русские названия, остальные сохранены в английском написании.

Таблица построена по WALS; поиск, сортировка, фильтры и пагинация работают через Tabulator.
